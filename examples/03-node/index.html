Example 03 — Node usage

Shows pulse app-layer listen() through a profile-local host binder from pulse.node.

The default portable path is still app.bind(). listen() only works because the active profile defines host: node(...).

03-node/index.ts
import { Pulse, defineConfig } from 'pulse';
import { node } from 'pulse.node';

const config = defineConfig((env) => ({
  profile: env('PULSE_PROFILE', 'local'),
  profiles: {
    local: {
      host: node({
        port: env.number('PORT', 3000),
        baseUrl: env('BASE_URL', 'http://localhost:3000')
      })
    }
  }
}));

const app = new Pulse({ config });
app.get('/health', async (ctx) => ctx.reply({ ok: true }));

const server = await app.listen();
console.log('Pulse Node server listening', server);
